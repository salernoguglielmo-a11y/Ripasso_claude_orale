# Ripasso Claude Orale

Web app per il ripasso dell'esame orale da avvocato, con focus su diritto penale, diritto civile e procedura penale.

## Stack
- React
- TypeScript
- Vite
- Tailwind CSS

## Funzioni già presenti
- dashboard iniziale
- filtri per materia e stato
- ricerca full-text sulle schede
- modalità simulazione orale con timer
- tracciamento locale del ripasso
- cambio stato delle schede
- dataset iniziale con schede giuridiche ragionate

## Avvio locale
```bash
npm install
npm run dev
```

## Stato del progetto
Questa repository contiene una prima base funzionante, pensata per essere ampliata progressivamente con nuove schede e maggiore profondità contenutistica per ciascuna materia.
